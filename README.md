# Sales Performance Dashboard

## Business Problem
Retail managers lacked visibility into sales performance across regions and product categories. This project analyzes sales data to uncover revenue drivers and profitability opportunities.

## Dataset
- Retail sales dataset
- 10,000+ transactions
- Fields: Sales, Profit, Category, Region, Customer, Order Date

## Tools Used
- Excel
- SQL
- Power BI
- Power Query

## Data Cleaning
- Removed duplicates
- Handled missing values
- Standardized product categories
- Created calculated measures

## Key Business Insights
- Technology category generated highest revenue.
- High discounts reduced profitability.
- West region showed declining sales trend.
- Top customers generated majority of revenue.

## Business Recommendations
- Reduce excessive discounts.
- Focus marketing on high-value customers.
- Improve underperforming regions.
